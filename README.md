yaza
====

yet another zazzle


## quickstart

1. `$ sudo apt-get install rubygems`
2. `$ sudo gem install sass`
3. `$ sudo npm install -g grunt-cli`
4. `$ git clone ...`
5. `$ cd <repos>/static/js/vendor/bootswatch-scss`
6. `$ sudo npm install grunt`
7. `$ sudo npm install`
8. `$ grunt build:flatly`
