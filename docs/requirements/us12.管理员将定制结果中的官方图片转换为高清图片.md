# 概述

为了安全起见， 甲方希望将所有的高清设计素材都放置在内网。 因此用户产生的定制结果， 除了用户自己上传的素材是原图以外， 其余由甲方提供的素材全部是低清版本， 在进行实际的打印生产之前， 必须要将其在内网中转化为高清素材。
转换的依据是, 发布在外网和低清版本设计素材和内网的高清版本设计素材必须同名。

# 浏览器支持

chrome, firefox

# 检查清单

针对ie9和其它任一浏览器生成的定制结果(这个定制结果应该至少包括文字，图片两种素材，　图片包括用户上传的，和系统自带的)执行如下操作：

1. 管理员登陆后， 进入“高清图转换”TAB页， 选择高清图所在本地目录，　然后选择要执行转换的设计稿文件，　点击转换。应当生成一个高清版本的设计稿。
2. 观察这个设计稿中的图片是否是高清的，　可以打开高清设计稿，放大后查看，　也可以直接查看文件大小是否显著增加。
